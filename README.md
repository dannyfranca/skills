# Skills

My reusable agent skills, published through [skills.sh](https://skills.sh/dannyfranca/skills).

## Install

```sh
npx skills add dannyfranca/skills
```

List or install one skill:

```sh
npx skills add dannyfranca/skills --list
npx skills add dannyfranca/skills --skill multi-shot-review
```

## Available skills

- `implement-issues-github` — Implement GitHub issues with reviews.
- `implement-tickets-local` — Implement local tickets with reviews.
- `micro-commits` — Create focused, incremental commits.
- `multi-shot-review` — Review risky changes through repeated Codex passes.
- `to-tickets-gh` — Publish tracer-bullet tickets as GitHub issues.
- `to-tickets-local` — Publish tracer-bullet tickets as local files.
- `worktrees` — Apply Git worktree setup conventions.
